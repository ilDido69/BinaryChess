This is a simple chess GUI.


Future features:

* Improve the GUI
* Add an engine
* Improve the move generation


Requirements:

* C++17 or higher
* CMake 3.16+
* SFML 3.0
* A C++ compiler (MSVC, GCC, Clang)

==============================================================================================================================================================


Easy installation:

* Clone the repository
* Install with vcpkg sfml 3.0
* Compile with cmake
